# Welcome to Portable Trust Infrastructure

Portable Trust Infrastructure (PTI) is an architectural category for systems that **produce, exchange, resolve, and consume trust** across institutional boundaries while preserving portability, explainability, consent, and governance.

This documentation corpus is **vendor-neutral**. It defines what it means to be PTI-compatible independent of any single implementation.
