# Bibliography

Reference materials supporting the Portable Trust Infrastructure (PTI) documentation ecosystem.

## Primary references

| Document | Description |
|----------|-------------|
| [PTI Whitepaper](./whitepaper) | Canonical long-form reference — trust fragmentation, PTI philosophy, signals, contexts, partner ecosystem, and roadmap |
| [PTI Specification v1.0](/pti/specification/v1.0/) | Normative architecture, APIs, schemas, security, governance, and privacy requirements |

## Standards and RFCs

| Reference | Relevance |
|-----------|-----------|
| [RFC 2119](https://datatracker.ietf.org/doc/html/rfc2119) | Key words for requirements in normative PTI documents |
| [RFC 7519](https://datatracker.ietf.org/doc/html/rfc7519) | JSON Web Token (JWT) structure for bearer and federation tokens |
| [RFC 7515](https://datatracker.ietf.org/doc/html/rfc7515) | JSON Web Signature (JWS) for assertions and webhooks |
| [ISO 8601](https://www.iso.org/iso-8601-date-and-time-format.html) | Date and time formats for events and reports |
| [ISO/IEC 27001](https://www.iso.org/standard/54534.html) | Information security management mapping (informative) |

## Citation format

When citing PTI documentation in academic or policy work:

> Portable Trust Infrastructure (PTI) v1.0 Specification. Trust documentation ecosystem. Available at: `/pti/specification/v1.0/`.

For the whitepaper:

> Portable Trust Infrastructure Whitepaper. Bibliography collection. Available at: `/pti/bibliography/whitepaper`.

## Contributing references

Operators and researchers **MAY** propose additional bibliography entries via the documentation repository when sources materially inform PTI design decisions (identity resolution, governance, interoperability, or emerging market trust patterns).
